# Privacy Policy for Student Retention Kit

**Last Updated:** [Current Date]

## Overview
Student Retention Kit is a Chrome extension developed exclusively for NUC University retention staff to monitor at-risk students and conduct outreach calls through the university's Canvas LMS and Five9 call center systems.

## Intended Use
This extension is designed **exclusively for authorized NUC University retention staff** and is not intended for:
- General public use
- Use by other educational institutions
- Use by students or unauthorized personnel
- Commercial purposes

## Data Collection

### What Data We Collect
- **Student Information:** Names, student ID numbers, phone numbers, Canvas user IDs of NUC University students
- **Academic Data:** Grades, missing assignments, course enrollment, and submission history from NUC University Canvas courses
- **Canvas Activity:** Page visits and assignment submissions within NUC University Canvas LMS (nuc.instructure.com)
- **Call Records:** Call dispositions and notes for student outreach (stored locally)

### How We Collect Data
- Data is imported from NUC University student information systems (CSV/Excel files)
- Academic data is fetched via NUC University Canvas LMS API using your authenticated NUC credentials
- Canvas activity is monitored only on the authorized NUC University Canvas domain (nuc.instructure.com)

## Data Storage
- **All student data is stored locally** in your browser using Chrome's storage API
- **No data is transmitted** to external servers except:
  - Authenticated API calls to NUC University Canvas LMS (nuc.instructure.com)
  - Authenticated API calls to NUC University Five9 call center (app-atl.five9.com)
  - Optional exports to Microsoft Office/SharePoint (user-initiated, for internal NUC reporting)

## Data Usage
- Data is used **solely** for NUC University student retention tracking and outreach management
- We **do not sell** or transfer student data to third parties
- We **do not use** student data for purposes unrelated to NUC University student retention activities
- We **do not use** student data for creditworthiness or lending purposes

## Data Access
- Student data is only accessible to **authorized NUC University retention staff** who are authenticated in Canvas and Five9
- Extension operates within the **same permissions** granted to you by NUC University's Canvas and Five9 systems
- No unauthorized access to NUC University student records

## Data Security
- Student data is protected by Chrome's built-in security features
- Authentication is handled by NUC University's Canvas and Five9 authentication systems
- No passwords or credentials are stored by the extension
- All data access requires valid NUC University credentials

## Data Retention
- Student data remains in local storage until:
  - You manually clear it using the extension
  - You uninstall the extension
  - You clear your browser data
- Data is subject to NUC University's data retention policies

## Third-Party Services
The extension integrates exclusively with NUC University authorized systems:
- **NUC University Canvas LMS** (nuc.instructure.com) - For student academic data
- **NUC University Five9** (app-atl.five9.com) - For call center functionality
- **Microsoft Office/SharePoint** (optional) - For internal NUC reporting and data export

All integrations require your existing authenticated NUC University sessions.

## FERPA Compliance
This extension is designed to comply with the Family Educational Rights and Privacy Act (FERPA) and NUC University's policies regarding student educational records:
- Access is limited to authorized university officials with legitimate educational interest
- Student data is used solely for retention and student success purposes
- No student data is disclosed to unauthorized parties

## User Rights
As an authorized NUC University staff member, you have the right to:
- Access stored student data via the extension interface
- Delete stored data by clearing storage or uninstalling
- Export data for internal NUC reporting purposes

## Children's Privacy
This extension is designed exclusively for NUC University staff members and is not intended for use by students or children under 13.

## Institutional Authorization
Use of this extension constitutes acceptance of:
- NUC University's acceptable use policies
- NUC University's data privacy and security requirements
- Your responsibilities as an authorized university official handling student records
