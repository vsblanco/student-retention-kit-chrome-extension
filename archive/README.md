# Archive Directory

This directory contains deprecated and unused files from the Student Retention Kit Chrome Extension.

## Unused Scripts

- **popupSearch.js** - Old search functionality that was replaced with inline search in popup.js. Not loaded in any HTML file.

## Unused Assets

Located in `unused-assets/`:
- **settingsicon.png** - Settings icon not referenced in any code
- **notification.png** - Notification icon not referenced in any code

These files are kept for historical reference but are not used by the extension.
